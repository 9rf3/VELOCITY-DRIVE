# VelocityDrive

A premium luxury supercar marketplace and showcase platform built with modern frontend technologies.  
Inspired by brands like Bugatti, Porsche, Lamborghini, and Koenigsegg.

---

## Overview

VelocityDrive is a cinematic automotive experience website focused on luxury hypercars and premium UI design.

The project is built to demonstrate:
- modern frontend architecture
- responsive layouts
- smooth animations
- premium UI/UX
- realistic product presentation

This is a frontend-focused concept project with immersive visuals and interactive user experience.

---

## Features

### Landing Page
- Cinematic hero section
- Fullscreen luxury car showcase
- Smooth scrolling animations
- Brand presentation sections
- Performance statistics

### Car Catalog
- Responsive car grid
- Brand filtering
- Search functionality
- Luxury-style cards
- Interactive hover effects

### Car Details Page
- Fullscreen image gallery
- Detailed specifications
- Performance visualization
- Animated statistics
- Interior showcase
- Engine sound interaction

### Compare System
- Compare two vehicles
- Performance comparison table
- Interactive comparison UI

### Favorites
- Save favorite vehicles
- LocalStorage support

### Responsive Design
- Mobile optimized
- Tablet support
- Desktop cinematic experience

---

## Tech Stack

- React / Next.js
- JavaScript
- Tailwind CSS / CSS Modules
- Framer Motion
- Swiper.js
